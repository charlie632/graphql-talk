# It all starts with a schema

A way to describe your data

```gql
type User {
  name: String!
  id: ID!
  orders: [Order]
}

type Order {
  selection: Int
  meal: String
}
```

---

# Define operations

```gql
type Query {
  getUser(id: ID!): User
  getUserOrders(userId: ID!): [Order]
}

type Mutation {
  addUser(user: UserInput): User
  addUserOrder(userId: ID!, selection: number, meal: String): Order
}
```

---

# How to call?

You write a query

```gql
query GetUser {
  getUser(id: "123abc") {
    name
    id
  }
}
```
